# Celestial Seal
An Authentication blilerplate code app, ready to be used in any application with authenticaion ready as we start developing it.

## Progress:

- Backend: Done ✅️
- Frontend: In progress ❌️

## Techs:

- Monorepo: Turborepo
- Backend: Nest.js
- Frontend: Next.js
- Database: PostgreSQL
- ORM: Prisma
- Tokens: JWT
